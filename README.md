# API Documentation

## Create Employee

Creates a new employee name.

- **URL:** `/employee`
- **Method:** POST
- **Headers:**
  - Content-Type: application/json
  - access_token: [Access Token]

### Request Body

```json
{
    "name": "example name"
}
